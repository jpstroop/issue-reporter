Creator:
Francis Kayiwa
Created:
2022-07-22 15:48:04
Status:
Open
Comments:
hackartisan at 2022-11-16 14:11:35 :

Notes from All-hands session which may be relevant: https://docs.google.com/document/d/1C_VeXTF7-9zImLq6j_0TJWtumSehDuJ4-svgISQRBlo/edit#heading=h.wsvjvyxj8wkh

hackartisan at 2022-11-16 14:14:23 :

Proposal: A ruby script that uses the github API to fetch security / dependabot alerts for all (or a subset of) PUL projects. For each alert, it creates a github issue in https://github.com/PrincetonUniversityLibrary/security/issues, prefixed with some kind of persistent ID. The issue holds a link to the dependabot info, and a list of all affected PUL apps. The issue serves as a central location for developers to discuss approaches to resolution.

The script runs weekly. It doesn't remove existing issues but at each run it ensures the list of affected apps is current.
