#486 Missing field for related identifier

Creator:
Bess Sadler
Created:
2022-10-13 12:36:55
Status:
Open
Comments:
Matthew J. Chandler at 2022-10-13 12:52:31 :

Mapping relation definitions from DSpace to DataCite can be tricky. In our ongoing curation, we interpret "dc.relation.isreferencedby" from DSpace (see https://wiki.lyrasis.org/display/DSDOC5x/Metadata+and+Bitstream+Format+Registries) as "IsCitedBy" in DataCite. There can be confusion because DataCite also has an "IsReferencedBy" relation, but "IsCitedBy" is the current recommendation, so that is what we use for relations between datasets and their corresponding papers (see https://support.datacite.org/docs/datacite-metadata-schema-v44-recommended-and-optional-properties#12b-relationtype).

Hannah H at 2022-10-13 15:51:07 :

(Note that (12.a) relatedIdentifierType would be inputted to the form because it's a mandatory property when using 12 RelatedIdentifier, but this does not explicitly show on the discovery page. Please see the wireframes for the Describe form for how this set of properties should appear in the Describe form.)
