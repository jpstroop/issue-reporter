Kevin Reiss

2022-05-17 14:58:09
Status:
Open
Comments:
Jane Sandberg at 2022-10-14 17:36:11 :

Noting that there is a honeypot on this form. Steps to exercise the honeypot:

  1. Open mailcatcher using the instructions in the README
  2. Go to https://catalog-staging.princeton.edu/feedback
  3. Fill out the form and submit
  4. Note that an outgoing email to libanswers appears in the mailcatcher.
  5. Go back to https://catalog-staging.princeton.edu/feedback
  6. Apply this CSS rule in your browser to make the honeypot field appear: .blacklight-feedback-hidden { display: block; }
  7. Fill out the whole form, including the honeypot field, and submit
  8. Note that the form gives you a success message, but no email appears in the mailcatcher.

So we need to focus on the spam is not fooled by the honeypot.
