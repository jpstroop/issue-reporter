Status:

Comments:
hackartisan at 2022-10-04 14:10:55 :

We accessed a database backup from earlier in the week. We sshed directly to the storage machine, rather than running grep over the mount, because the file was 120G.

LC_ALL=C grep -A 2000 "COPY public.roles_users" [name of file]

Once we had the list of IDs, we put them into an array and ran the following in the rails console:

admin_role = Role.find_or_create_by(name: "admin")

admin_users.each do |user_id|
  u = User.find(user_id)
  next if u.roles.include? admin_role
  puts "Making #{u} an admin"
  u.roles << admin_role
  u.save
end
