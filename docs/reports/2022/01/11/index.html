Following notes in https://github.com/pulibrary/orangelight/issues/2821#issuecomment-1009118329 and @itmaybejj recommendation from the developer.mozilla page Warning: Be careful when marking up links with the button role. Buttons are expected to be triggered using the Space or Enter key, while links are expected to be triggered using the Enter key. In other words, when links are used to behave like buttons, adding role="button" alone is not sufficient. It will also be necessary to add a key event handler that listens for the Space key in order to be consistent with native buttons. need to complete the 3rd checkbox before merging.
