#182 ResourceType and resourceTypeGeneral

Creator:
Hector Correa
Created:
2022-06-27 19:08:33
Status:
Open
Comments:
Hannah H at 2022-06-27 19:58:45 :

@hectorcorrea --- For (10) ResourceType, we may need to discuss with the group on 7/8. We might not want the depositor to guess on what to enter for this field. For (10.1) resourceTypeGeneral, yes it should be a dropdown list. However, we might want to hide some of the choices that aren't deposit options for the research data repository. For example: Book, Book Chapter, Conference Proceeding, Dissertation, Journal, Journal Article, Peer Review, Preprint... possibly others, but let's ask the larger group.

10.1 resourceTypeGeneral should visibly appear under required for both depositor and admin/curators. 10 ResourceType I'm looking closer at and want more feedback from others (including you) if possible.

Hannah H at 2022-06-27 20:04:29 :

@hectorcorrea This documentation for creating a DOI via Fabrica/Datacite appears to show that only 10.1 resourceTypeGeneral is required on the form. See: https://support.datacite.org/docs/field-descriptions-for-form

Hannah H at 2022-06-27 20:17:21 :

Here are some examples of the XML code from the Datacite 4.4 help (note: I put weird spaces between characters, so Github wouldn't render to regular text.):

Example software: < resourceType resourceTypeGeneral ="Software" >Software < /resourceType >

Example a simple dataset: < resourceType resourceTypeGeneral = "Dataset" >Dataset < /resourceType >

Example for video as ResourceType: < resourceType resourceTypeGeneral = "Audiovisual" > narrated video

Example for a workflow ResourceType: < resourceType resourceTypeGeneral = "Workflow" > Software < /resourceType >

Hannah H at 2022-06-27 20:22:48 :

@hectorcorrea Looking at the examples for software and a simple dataset, we might simply duplicate what is chosen from the controlled list (10.1 resourceTypeGeneral) to the free text box (10 resourceType) and only let the admin/curator change the free text (10 resourceType), as may be needed in some cases. What do you think?
