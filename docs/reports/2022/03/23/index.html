#604 Export bitstreams via command line

Creator:
Bess Sadler
Created:
2022-02-28 16:08:52
Status:
Open
Comments:
Hector Correa at 2022-03-23 19:29:10 :

A possible way to handle this would be to fetch the bitstream info from DataSpace via the API and then download the files via cURL. The API provides the name of the file and even an MD5 to check for integrity.

Here is a proof of concept Ruby script to create a Bash script to download the files for a given collection: https://gist.github.com/hectorcorrea/d1b5e4f9d3a1de4a68114a194b0804f4

$ ruby ruby getbitstreams.rb > get1061.sh
$ chmod u+x get1061.sh
$ ./get1061.sh

Since the files will be downloaded to a folder named after the handle of the DataSpace item, these folders could be uploaded to Globus via the Globus Connect Personal.
