This is the transformation workflow from the days of the migration (SVN readme):

  1. Create container entities from dsc2 and classic containers: svn://lib-svn.princeton.edu/libsvn/trunk/rbscXSL/ASpace tools/get-containers.xquery svn://lib-svn.princeton.edu/libsvn/trunk/rbscXSL/ASpace tools/classic2dsc2_containers.xq

  2. Transform EADs: svn://lib-svn.princeton.edu/libsvn/trunk/rbscXSL/ASpace tools/ASpacedocontainers.xq svn://lib-svn.princeton.edu/libsvn/trunk/rbscXSL/ASpace tools/ASpacedocleanup.xq

  3. MUST run after EAD transformation: Extract Agents namesA-F and namesG-Z with get-agent-info-and-delete.xq and delete names, bioghists from EAD; Remove empty elements with remove-empty-agent-wrappers.xq

  4. Create CSV from namesA-F, namesG-Z using names2csv.xq

  5. Validate: svn://lib-svn.princeton.edu/libsvn/trunk/rbscXSL/ASpace tools/validate.xquery (5 min), svn://lib-svn.princeton.edu/libsvn/trunk/rbscXSL/ASpace tools/validate-containers.xquery
