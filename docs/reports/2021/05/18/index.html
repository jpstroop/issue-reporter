In order for SCSB to work with PUL owned items that are shelved in a remote_storage location when the request is placed it needs to:

  1. Place a SCSB Physical Item Request for the item in question
  2. If that is successful, place an Alma hold request for the same item. These requests should be identical to the hold requests that will be placed in Alma when this ticket is complete: https://github.com/pulibrary/requests/issues/933.

Materials owned by SCSB partners will still only need a single SCSB Physical item request placed.
