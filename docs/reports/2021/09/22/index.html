#932 Look into how theming works in Spotlight 3.

Creator:
Trey Pendragon
Created:
2021-09-10 14:38:19
Status:
Closed
Comments:
Shaun Ellis at 2021-09-22 20:13:47 :

Simple documentation of theming: 1. Add thumbnail image named as {theme-name}_preview.png to themes directory for each theme. (example: app/assets/images/spotlight/themes/default_preview.png) 2. Create stylesheets for each theme with a naming convention of application_{theme-name}.scss (note: application.scss is the default theme) 3. Remove *= require_tree . line from application.scss 4. Add Rails.application.config.assets.precompile += %w( application_{theme-name}.css ) to /config/initializers/assets.rb 5. Add Spotlight::Engine.config.exhibit_themes = %w[{theme-name} default] to /config/initializers/spotlight_initializer.rb
