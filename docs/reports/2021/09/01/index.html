#2695 Exporting Bookmarks as a CSV returns the wrong ID number

Creator:
Esmé Cowles
Created:
2021-08-31 14:57:06
Status:
Open
Comments:
Christina Chortaria at 2021-09-01 15:00:48 :

https://techcommunity.microsoft.com/t5/excel/when-i-paste-a-long-number-excel-turns-it-into-scientific/m-p/78489/highlight/true#M1171

Excel operates with 15 digits. Any digits beyond will change to 0. One possible solution is to add 'id' in front of the id number or add one single ' (quote) in the beginning of the id or single quotes around the id. Having single quotes around the id does not prevent the user to double click in the cell and copy only the id number. It really depends on how they use this exported csv file.

Christina Chortaria at 2021-09-01 15:18:52 :

The other option is to import the csv file using the excel wizard:

Data -> From Text -> (select the csv file) -> click Get Data -> Select Delimited (click Next) -> Select Comma (click Next) -> (Under column data format) Select Text (click Finish).

Esmé Cowles at 2021-09-01 16:10:00 :

@christinach I like putting the id in single quotes — that doesn't require special action for the user (which I think would be hard to communicate and unlikely to be followed), and it seems like it would keep the value usable in Excel and other applications.

Christina Chortaria at 2021-09-01 16:21:34 :

Ok @escowles will do that. Thanks!

#76 Redirect PUDL URLs

Creator:
Esmé Cowles
Created:
2021-01-06 15:28:05
Status:
Open
Comments:
Cliff Wulfman at 2021-09-01 14:11:49 :

New Strategy

Based on meeting btw @escowles and @cwulfman, we will pursue an agile strategy: in the first pass, simply direct all the items in a PUDL collection to its corresponding DPUL collection or finding aid (item -> collection); a second pass can redirect item -> item. This allows us to sunset the PUDL platform more quickly (the top priority).

Begin with GNIB and Latin American Posters, which have the largest number of item-level arks in PUDL.

Discussion

There are, probably, PUDL collections and/or items that don't map directly to DPUL collections or finding aids. There are, probably, few of these, however.
