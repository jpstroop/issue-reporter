#68 Create AlmaKbart Job

Creator:
carolyncole
Created:
2021-02-17 19:33:50
Status:
Open
Comments:
Peter Green at 2021-10-08 22:13:52 :

A Nov 2018 email message from Robert McGrath at Anywhere Access lists just 3 required KBART fields:

  • print_identifier - in Alma export
  • online_identifier - in Alma export
  • interface_name - NOT in Alma export

Since 2018 we've been providing filenames (File.basename minus dates) as the interface_name. It looks like we'll probably need to run the same sort of script: 1. configure a weekly job that exports KBART for portfolios to a directory on lib-sftp 2. loop over the exported files, gathering their data and filenames 3. append filenames to existing fields (as the required interface_name) in a single combined csv 4. zip and push the resulting file (using their requested naming convention) to ftp.readcube.com. E.g. EXPORT_PORTFOLIOS_PRINCETON_SFX_20210719064145.zip

@kevinreiss is it worth checking with Anywhere Access to see whether their requirements have changed?
