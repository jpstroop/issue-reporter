#497 Map 2021 Senior Thesis items from certificate programs to their DataSpace collections.

Creator:

Kate Lynch at 2021-10-21 18:27:34 :

As stated in the linked issue from 2020, the following action run on the server and then at the dspace CLI prompt resolves this issue:

Load the JRuby terminal: ``` $ source $HOME/.rvm/scripts/rvm $ cd ~/pulibrary-src/dspace-cli $ bin/dspace-irb $eperson Using /dspace Loading jars Loading /dspace/config/dspace.cfg INFO: Loading provided config file: /dspace/config/dspace.cfg INFO: Using dspace provided log configuration (log.init.config) INFO: Loading: /dspace/config/log4j.properties Starting new DSpaceKernel DB jdbc:postgresql://localhost:5432/dspacedb, UserName=dspacedb_user, PostgreSQL JDBC Driver jruby-9.2.13.0 :001 >


In the JRuby terminal:

certhandle = '88435/dsp01c247ds15b' certcollection = DSpace::CLI::SeniorThesisCollection.findforhandle(certhandle) query = DSpace::CLI::SeniorThesisCommunity.findbyclassyear('2021')

certprogram = 'Theater Program' subquery = query.findbycertificateprogram(certprogram) collectionhandle = '88435/dsp01c247ds15b' items = [] subquery.results.each do |item| addtocollection = true item.collections.each do |ic| addtocollection = false if ic.handle == collectionhandle end items << item if addto_collection end

items.each do |i| certcollection.additem(i) cert_collection.update end

certcollection.index `` Wherecerthandleis the certificate program ark to be mapped (consult the table above), the matchingpu.certificatevalue from the item's metadata (for exampleTheater Program,Creative Writing Program),collectionhandleis the handle for the DataSpace collection where the item should appear after this code is executed (see table in ticket description above), and the parameter in the call tofindbyclassyear` matching the current year.

I ran this code against the following item, after adjusting the class year to 2021, and adding a pu.certificate field instance with the value Theater Program: https://dataspace-dev.princeton.edu/handle/99999/fk4321d72q

I also adjusted the following item on staging (imported in 2020) to be in the class year 2021, with the pu.certificate value of Theater Program.: https://dataspace-staging.princeton.edu/handle/88435/dsp01xw42nb94p

We should be safe to run this on production ASAP.
