Creator:

Comments:
Kim Leaman at 2021-02-12 17:12:43 :

Brittany set up to work directly on this, as well as the digitization. set up meetings and confirmation with Jennifer Baxmeyer of MS and Peter Green of EUS. Branch Library Ongoing project. Records now in MARC, but suppressed:

Hey Kim! Thanks again for organizing the meeting. I’m sorry we didn’t get through everything we were hoping to get through! 

So I’ll tell you why the records were entered into the catalog suppressed. When Peter and Mark were creating the loader to get the bib records into Voyager, Mark asked why we were adding bib records if we didn’t have anything for people to find. Jennifer and Peter decided that they would load the records as suppressed until there was images to connect to the bib in figgy.
I did bring up that we have the physical items and the bib records somehow need to also show that, but it didn’t go further. I’m not sure how we setup the physical items to connect to the bib and MFHD records. There are about 20-50 individual songs in a volume. Right now there is one main record for the valva collection and then barcodes for each volume within the collection. This is the record for the collection in the catalog: https://catalog.princeton.edu/catalog/2434779   It currently just points to the google document that was used to create the individual bib records for each song within the collection. So if we can find a way to make sure that each bib record for the songs are linked to the volume that’d be great, but I don’t know the best option. We would need to add a barcode to each song, but there are often two songs on one piece of paper within the volume. (it’s a bit confusing). I can pull out examples of everything if that helps.
