#453 Update Task Pool entries for member Lib_DigPub_Reviewers and Lib_DigPub_Approvers

Creator:
James R. Griffin III
Created:
2021-06-29 14:23:27
Status:
Open
Comments:
James R. Griffin III at 2021-06-29 15:05:59 :

The following proved effective at ensuring that all Task Pool entries should be created:

java_import(org.dspace.eperson.Group)
java_import(org.dspace.workflow.WorkflowItem)

name = ""
group = org.dspace.eperson.Group.findByName(DSpace.context, name)
members = org.dspace.eperson.Group.allMembers(DSpace.context, group)

items = []
members.to_a.each do |member|
  objects = org.dspace.workflow.WorkflowItem.findByEPerson(DSpace.context, member)
  items << objects
end

members.to_a.each do |member|
  objects = org.dspace.workflow.WorkflowItem.findByEPerson(DSpace.context, member)
  delta = items.uniq - objects
  workflows = objects.to_a.map { |obj| DSpace::CLI::WorkflowItem.new(obj) }
  workflows.each { |workflow| workflow.add_task_pool_user(member.getEmail) }
end
