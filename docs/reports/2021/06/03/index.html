#7 Automatically generate "next barcode" via CODABAR.

Creator:
Trey Pendragon
Created:
2021-06-01 16:36:55
Status:
Closed
Comments:
Trey Pendragon at 2021-06-03 19:57:23 :

To get the next barcode: Remove the check digit Add 1 to the number Sum all the digits, multiply by 2. If it's greater than 9, subtract 9. Take the last digit of the result. If it's zero, the check digit is zero. Otherwise, 10 - digit is the check digit.

Try it out:

32101113342891 3210111334289 + 1 = 3210111334290 3+2+1+0+1+1+1+3+3+4+2+9+0 = 30 * 2 = 60 -9 = 51, 10-1 = 9.

Result: 32101113342909

Yay!

This is Luhn's algorithm I guess.
