#809 Incremental index from Alma

Creator:
hackmastera
Created:
2020-09-25 15:02:21
Status:
Open
Comments:
hackmastera at 2020-10-15 19:51:53 :

Proposal:

  1. Move work from https://github.com/pulibrary/alma_experimentation/pull/8/files and https://github.com/pulibrary/alma_experimentation/pull/33/files into marc_liberation.
  2. Update the webhook handler. If the webhook action/event indicates an incremental dump job is complete, enqueue the event body to an AWS SQS FIFO queue.
  3. Use the sidekiq cron gem https://github.com/ondrejbartas/sidekiq-cron to schedule a job to run every 15 minutes as follows:
    1. Is this job is already running? (example, see option 3)
      1. If so, terminate.
      2. If not, check the queue and if there's an event invoke the service that will create the Dump and Event objects, copy the files, and index them.

This should ensure that only one incremental job is processed at a time, and that they are processed in order.
