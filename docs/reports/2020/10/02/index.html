Hi there, I wanted to add some info to the description above and request some additional features. In addition to generating the AbID and barcodes, we would need the tool to go into ASpace and plug them in to the Top Container records: replacing traditional box numbers that were entered during processing and adding the barcodes.

Additionally I have been struggling with the fact that you can't add container profiles or locations to Top Containers created in ASpace using Rapid Data Entry or the spreadsheet importer. We would likely need to add both of these to every Top Container record, and so I'm wondering if this is something we could add to the tool, since the data is going to the same place in ASpace (Top Container records).

I could see this working in the tool because a user would be selecting a box size (which is what AbIDs are based on), and additionally they could add the location as well. there could potentially be more than one location added per collection.
