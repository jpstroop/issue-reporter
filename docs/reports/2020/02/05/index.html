#1263 Provision solr 8 boxes for use by catalog-staging

Creator:
hackmastera
Created:
2020-01-31 15:46:08
Status:
Open
Comments:
hackmastera at 2020-02-05 21:18:19 :

Migrating solr data to a new znode.

  1. On each solr box, create a copy of /etc/defaults/solr.in.sh to /etc/defaults/newsolr.in.sh. Update the ZK_HOST param to append the new znode. When it's time to edit this config file, you can just swap in the copy.
  2. Stop solr on all boxes.
  3. On one box only:
    1. Start a tmux session.
    2. cd /opt/solr
    3. bin/solr zk mkroot /solr7 -z lib-zk-staging1:2181,lib-zk-staging2:2181,lib-zk-staging3:2181
    4. bin/solr zk cp -r zk:/clusterstate.json zk:/solr7/clusterstate.json -z lib-zk-staging1:2181,lib-zk-staging2:2181,lib-zk-staging3:2181
    5. bin/solr zk cp -r zk:/aliases.json zk:/solr7/aliases.json -z lib-zk-staging1:2181,lib-zk-staging2:2181,lib-zk-staging3:2181
    6. bin/solr zk cp -r zk:/live_nodes zk:/solr7/live_nodes -z lib-zk-staging1:2181,lib-zk-staging2:2181,lib-zk-staging3:2181
    7. bin/solr zk cp -r zk:/overseer zk:/solr7/overseer -z lib-zk-staging1:2181,lib-zk-staging2:2181,lib-zk-staging3:2181
    8. bin/solr zk cp -r zk:/overseer_elect zk:/solr7/overseer_elect -z lib-zk-staging1:2181,lib-zk-staging2:2181,lib-zk-staging3:2181
    9. bin/solr zk cp -r zk:/collections zk:/solr7/collections -z lib-zk-staging1:2181,lib-zk-staging2:2181,lib-zk-staging3:2181
    10. bin/solr zk cp -r zk:/configs zk:/solr7/configs -z lib-zk-staging1:2181,lib-zk-staging2:2181,lib-zk-staging3:2181
    11. Check the new znode contents: bin/solr zk ls /solr7 -z lib-zk-staging1:2181,lib-zk-staging2:2181,lib-zk-staging3:2181
    12. Delete those data from their old locations.
    13. bin/solr zk rm -r /clusterstate.json -z lib-zk-staging1:2181,lib-zk-staging2:2181,lib-zk-staging3:2181
    14. bin/solr zk rm -r /aliases.json -z lib-zk-staging1:2181,lib-zk-staging2:2181,lib-zk-staging3:2181
