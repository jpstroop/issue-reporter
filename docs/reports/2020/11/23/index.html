#147 Generate regularly-scheduled DataSet reports for downloading Item Bitstreams from DataSpace

Creator:
James R. Griffin III
Created:
2020-08-12 13:23:58
Status:
Open
Comments:
Kevin McElwee at 2020-11-23 15:12:34 :

Advice / script from Atmire: ```bash

put your handle here

handle="88435/dsp01jq085k007"

we need the item id...this gets it from the db

if eventually attempting this remotely without the access to psql, you can utilize

the Admin Edit UI to retrieve the internal item id https://share.getcloudapp.com/jkuYr2kp

id=$(psql -c "copy(select resourceid from handle where handle = '$handle' and resourcetype_id = 2) to stdout with csv")

total downloads

solr-count statistics "type:0 AND owningItem:$id"

downloads by ip

solr-facet statistics "type:0 AND owningItem:$id" ip

total page views

solr-count statistics "type:2 AND id:$id"

page views by ip

solr-facet statistics "type:2 AND id:$id" ip ```

Other facet queries that can be done for the above are "userAgent", "countryCode", "dns", and "isBot" (just replace "ip" at the end of the queries above with these instead). You can also put additional ANDs in there, to limit by isBot:false, e.g.: solr-facet statistics "type:2 AND id:$id AND isBot:false" countryCode For bitstream downloads, you can also facet on "id" to break it down by specific bitstream (if some items have multiple files in them).
