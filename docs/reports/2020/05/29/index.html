Query for publications will be something like:

search publications where research_org_names ="Princeton University" or research_orgs="grid.16750.35" return publications[all]

and datasets: search datasets where research_orgs="grid.16750.35" return datasets[all]

The response is verbose("all"), but that can be tailored. Next step is to figure out paging.

Eliot Jordan at 2020-05-29 17:05:21 :

For paging, use limit for page size and skip for offset.

search datasets where research_orgs="grid.16750.35"
return datasets[all] limit 100 skip 100

Total record count is returned in each request in the _stats property:

 "_stats": {
        "total_count": 1916,
        "limit": 100,
        "offset": 100
    }
