#132 Privilege description of materials over container information

Creator:
Amanda Ferrara
Created:

Status:
Open
Comments:
Shaun Ellis at 2020-03-30 16:29:27 :

faithc at 2020-03-30 18:11:32 :

@sdellis Hi Shaun- this relates to something the previous WG had discussed. This may be more of a design-related issue than one related to functionality(?); but as with the current site, we would like to privilege description over container information. So, rather than having description display as such: "Folder 2: Letter from Abraham Lincoln to James Gordon Bennett, 1865," where the container information is displayed first, we'd like the order to be reversed. (This is the collection I'm using as an example: https://pulfalight-staging.princeton.edu/catalog/C0094.) This method also makes sense since, as you know, the description of an intellectual grouping can be associated with several containers.
