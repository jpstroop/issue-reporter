#4233 As a user, I want to be able to ingest non-catalogued items into Figgy

Creator:
Cliff Wulfman
Created:
2020-08-31 15:41:43
Status:
Open
Comments:
Trey Pendragon at 2020-08-31 18:35:12 :

We need to talk about this in a longer form format. Maybe @cwulfman or @kelea99 can put together a meeting? According to the Archival Description in Figgy working group, all items which have a finding aid should have item-level finding aid metadata. In cases where there isn't, a data reconciliation project should be undertaken (much as was done for western americana.) You can find the report here: https://docs.google.com/document/d/1pkhMDZcumPWgfSBiFzkTzaEVpuvwTMJMHuzW-UoDrCc/edit

As a historical note, we've put a lot of effort into trying to not turn Figgy into a cataloguing system, largely because good ones exist already and the danger is that we'd just make a bad one that doesn't reflect the training and expertise of our cataloguers. This has shown to be true with some metadata being entered into Ephemera, for instance.
