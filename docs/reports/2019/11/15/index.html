Tooling is set up to then present those hashes via solr documents in metadata divs on the show page.

Final step will be to hide / show those divs in response to click events on the tabs which currently control the viewer display.

We'll also need to explicitly test that the third case, where there are no versions but we want to still display the metadata, remains true.
