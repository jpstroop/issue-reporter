I’m afraid I have bad news on this:

In other words, for the functionality you are requesting (the drop-down menu), we need to choose between the EAD schema and the local schema. The only workaround would be to go to a fully local schema that incorporates our local validation rules. That would be very hard to maintain though, so I don’t recommend it.

It’s possible to associate multiple schemas with a document type, if that would be helpful. This would at least make error messages from both schemas show up at the same time. Let me know if you want me to walk you through the setup.
