#2151 Update list of indigenous languages

Creator:
Jane Sandberg
Created:
2023-05-05 18:59:54
Status:
Open
Comments:
M Chen at 2023-05-05 20:36:37 :

A bib can be tagged as in Indigenous Languages (Western Hemisphere) if it meets any of the following criteria:

  1. The language code in its fixed field 008 (35-37) matches an entry in (2023-05)Indigenous language (Western hemisphere) names and codes > Tab MARC code
  2. A code in 041 $a, $d, $e etc. matches an entry in the Tab "MARC code"
  3. A code in 041 $a, $d, $e etc. matches an entry in the Tab ISO code whereas 041 has a source specified as $2 iso639-3
  4. A subject heading in 650 left-matches an entry in the Tab LCSH whereas 650 has a form subdivision $v Texts
