Shaun Ellis at 2023-05-10 19:57:17 :

The consensus is that, at a minimum, there will be some system notification to the user that the issue was reported. This may be a flash alert or animation that confirms it was successfully reported (or not). Th state of which files have been reported would not be saved on the server.
