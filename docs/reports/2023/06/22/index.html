Trey Pendragon at 2023-06-22 23:03:07 :

Thinking through how to preserve everything without re-preserving things if not necessary.

To preserve everything:

Go through every resource Should it be preserved? Skip if no Does it have a PreservationObject? If no, queue PreserveResourceJob Does the PreservationObject have all the binarynodes and metadatanode it should have? Queue a forced PreserveResourceJob if no. Do the checksums of the binarynodes match the stored checksum upstream? (Check metadata, don't actually run a CloudFixityJob, to save money.) If no, force PreserveResourceJob. Do the checksums of the metadatanode match the stored checksum upstream? (Check metadata, don't actually run a CloudFixityJob). If no, PreserveResourceJob with no force flag.

Other considerations:

Should we be thinking about parents/children here? Maybe we should run the parents first? An easy first step might be to just save all parent resources. That'll ensure anything that doesn't have a PreservationObject gets one.
