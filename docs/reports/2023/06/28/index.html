#5875 Migrate Derrida Source Metadata Identifiers

Creator:
Trey Pendragon
Created:
2023-06-19 18:01:09
Status:
Closed
Comments:
Trey Pendragon at 2023-06-28 00:02:55 :

These are done. I ended up doing it with a script because I think the proper feature here is something @hackartisan has been talking about, which is allowing bulk edit via CSV upload, and I didn't want to start that in a vacuum.

require 'csv'
csp = ChangeSetPersister.default
rows = CSV.read(Rails.root.join("tmp", "DerridaUnsuppressedCID+MMSID+ARK.csv"), headers: true)
rows.each do |row|
  row = row.to_h
  next unless row[row.keys.first] && row["MMS ID"]
  resources = csp.query_service.custom_queries.find_by_property(property: :source_metadata_identifier, value: row[row.keys.first])
  resources.each do |resource|
    change_set = ChangeSet.for(resource)
    change_set.validate(source_metadata_identifier: row["MMS ID"])
    puts "Changing #{row[row.keys.first]} to #{row["MMS ID"]}"
    csp.save(change_set: change_set)
  end
end
