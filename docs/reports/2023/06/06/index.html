After discussing with John K., I understand this better. The grayed-out memory setting (40 GB in the screenshot above) is the maximum available RAM for the VM. That setting can only be changed with the VM is powered off. The Limit setting can reduce the available RAM for a VM below the main Memory setting, and can be changed without powering the VM off. So we should consider setting Memory to something larger than we think we need, then using Limit to find the optimum setting.
