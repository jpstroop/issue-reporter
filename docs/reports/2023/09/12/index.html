Message for stakeholders

The DACS team is excited to invite you to test a new release of the catalog running Blacklight 8 in our catalog-qa environment. This release of Blacklight is focused on improving application preformance and implementing compatibility with newer versions of the Solr indexing server we use to index our catalog data. The release strives to implement the same search functionality the current catalog has for our basic and advanced search features.

The new release will be available for the new two weeks in the QA environment. To test you need to do the following.

  1. Connect to Global Protect if you are off campus or on on campus wifi.
  2. Visit https://catalog-qa.princeton.edu.
  3. Utilize the system, particularly the search features both basic and advanced.
  4. Please report errors or questions to this email thread or in slack on either the catalog or orangelight channels.
  5. When reporting an error please try and capture the URL and a screenshot (if possible) where you encountered something unexpected. Try and describe the issue in as much detail as you can.

Thanks for assisting with evaluating this release,

Regards, DACS Team
