Looks like this is the gem that lets us authenticate to orcid and get a token: https://github.com/datacite/omniauth-orcid

We need to register our application (there is a staging and a production environment). For full access we need to be organization orcid members, and I've asked on slack for confirmation that this is so.

We need to provide a contact email address and a secondary email address, as well as the URLs the application will use and the redirect urls that omniauth will use. Form is here, for reference: https://info.orcid.org/register-a-client-application-production-member-api/

The page also says we should expect some lead time:

Note: This process is not fully automated. There may be a delay before we respond to you with credentials or a demo request after you complete this form.

There appears to be pretty good documentation for how to configure this in rails: https://github.com/datacite/omniauth-orcid#rails
