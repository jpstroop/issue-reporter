pulibrary/ExcelAlmaLookup

#1 Suggestion: Search and retrieve barcodes and PIDs

Creator:
etaft
Created:
2023-11-16 14:37:12
Status:
Open
Comments:
tventimi at 2023-11-20 19:35:14 :

Hello, and apologies for the delay in replying. I am glad you are finding the tool helpful! It should already be possible to search by barcode and PID. In the main dialog for the tool, if you click the "Additional Fields" button, you can see the full set of Alma indexes. If you search for "barcode", you will see the name of the barcode index in your Alma instance (for Princeton, it is "alma.barcode"). And if you search for "pid", you should see index names for a few different types of PIDs. If you select the one you want and click the "Select" button, it should populate the search field with the appropriate code.

As far as retrieving these values, certain PIDs may be visible in the Availability fields. For example, for electronic resources, the Portfolio PID can be found in field AVE$8. What kind of PIDs are you looking for? If you let me know your institution's SRU URL, I can see what data is found in the Availability fields.

Retrieving barcodes cannot be done in the current version of the tool, but I am working on this feature and should be able to release it soon. However, for this to work, your institution must enable a setting called "ISO Holdings". If you want to request that your Alma admin enable this feature, you can send them the following documentation link.

https://knowledge.exlibrisgroup.com/Alma/Product_Documentation/010Alma_Online_Help_(English)/090Integrations_with_External_Systems/030Resource_Management/190SRU_SRW_Search (See the section "Holdings Options (ISO 20775)")

If your admin enables this setting, please let me know, and I can test out barcode retrieval on your catalog to make sure it will work for you.

Thank you for your interest!

Tom Ventimiglia Princeton University Library
