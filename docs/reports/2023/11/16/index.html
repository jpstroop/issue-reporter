#3778 Determine where submissions should be collected for the biased search results form

Creator:
Kate Lynch
Created:
2023-10-09 19:27:25
Status:
Open
Comments:
Kate Lynch at 2023-11-16 18:53:51 :

According to the proposed workflow in this document, submissions would be collected in a dedicated LibAnswers queue for reporting biased search results, in line with the workflows used by IRMWG. This queue would include representation from DLS, DACS, RDSS (likely entire teams). These representatives would be the first to triage form submissions and then loop in appropriate parties (such as IRMWG, AUX, DSSG, and more) to respond to issues.

It has been proposed that reviewing the LibAnswers queue and claiming unclaimed submissions would be part of the duties of the runner for each team (see runner documentation for RDSS for an explanation of the runner role).
