#354 One Solr node seems to be down

Creator:
Hector Correa
Created:
2022-11-01 14:29:46
Status:
Closed
Comments:
Hector Correa at 2023-03-23 16:02:35 :

We could not get a new replica to work on the existing pdc-discovery-production collection. We tried deleting the existing replicas and re-creating them in lib-solr-prod4/5/6 to no avail.

Instead we created a new Solr collection called pdc-discovery-production-2 with two replicas and that worked perfectly. We then re-indexed the data by running the reindex rake task with SOLR-URL env pointing to http://lib-solr8-prod.princeton.edu:8983/solr/pdc-discovery-production-2.

UPDATE on Mar/2023 To create the new collection we do it via the Solr Admin "Add Collection". There is already a config set for pdc-discovery-production that we use to create the new one.

Once we confirmed the data was reindexed in pdc-discovery-production-2 we deleted the old pdc-discovery-production collection and created an alias so that the new pdc-discovery-production-2 collection is also known as pdc-discovery-production, this way the PDC discovery code and configuration does not need to be updated.
